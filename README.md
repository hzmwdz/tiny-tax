# TinyTax

## Install

```bash
composer require hzmwdz/tiny-tax
```

## Publish

```bash
php artisan vendor:publish --provider="Hzmwdz\TinyTax\TinyTaxServiceProvider" --tag="migrations"
php artisan vendor:publish --provider="Hzmwdz\TinyTax\TinyTaxServiceProvider" --tag="translations"
```
